
Setting Label to Each Node which should be string only :
====================================================== 
Show Node Label :  kubectl get nodes --show-labels
Set Label to Node 1 : kubectl label nodes worker-1 priority=1
Set Label to Node 2 : kubectl label nodes worker-2 priority=2
Edit/update any label to any node : kubectl edit nodes worker-2

kind: ReplicaSet
metadata:
  name: replicaset-name
  labels:
    env: replicalabel
    type: prod
    priority: high
spec:
  replicas: 3
  selector:
    matchLabels:
      type: prod
      priority: high
  template:
    metadata:
      labels:
       type: prod
       priority: high
    spec:
     containers:
     - name: nginxcontname
       image: nginx
